## Аутентификация Twitch

При первом визите нажмите кнопку `Authenticate with Twitch`, чтобы выполнить аутентификацию Twitch.  
Отображаемый список будет базироваться на аутентифицированном пользователе.

## Follower List

Отображается список пользователей, которые в настоящее время подписаны на ваш канал.

## New followed List

Отображается список пользователей, которые начали подписываться на ваш канал с даты `Last checked`.

## Unfollowed List

Отображается список пользователей, которые отписались от вашего канала с даты `Last checked`.

## Дифференциальное обновление

Нажатием кнопки `CHECK DONE` дата `Last checked` будет обновлена. Это затронет `New followed List` и `Unfollowed List`.

## Обновление списков

Нажмите кнопку `REFRESH LISTS`, чтобы снова получить список подписчиков и обновить списки.  
Используйте эту функцию во время прямой трансляции или всякий раз, когда вы хотите проверить переходы подписчиков в реальном времени.  
Из-за ограничений API проходит несколько минут задержки, прежде чем вы сможете снова нажать кнопку.

## Фильтрация пользователей

Вы можете выполнять фильтрацию пользователей и поиск из поля поиска с надписью `Search...`.  
Все текстовые колонки, такие как дата, когда пользователь подписался на канал, и отображаемое имя, являются доступными для поиска.
