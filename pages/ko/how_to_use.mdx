## Twitch 인증

첫 방문시 `Authenticate with Twitch` 버튼을 클릭하여 Twitch 인증을 수행하세요.
표시된 목록은 인증된 사용자를 기반으로 합니다.

## Follower List

현재 채널을 팔로우하는 사용자의 목록이 표시됩니다.

## New followed List

`Last checked` (마지막 확인) 날짜 이후 채널을 팔로우하기 시작한 사용자의 목록이 표시됩니다.

## Unfollowed List

`Last checked` (마지막 확인) 날짜 이후 채널을 언팔로우한 사용자의 목록이 표시됩니다.

## 차별 업데이트

`CHECK DONE` 버튼을 누르면 `Last checked`의 날짜가 업데이트됩니다. 이것은 `New followed List`와 `Unfollowed List`에 영향을 줍니다.

## 목록 새로 고침

`REFRESH LISTS` 버튼을 눌러 팔로워를 다시 검색하고 목록을 업데이트하세요.
이 기능은 라이브 스트림 중이거나 팔로워 전환이 실시간으로 확인하고 싶을 때 사용하세요.
API 제한으로 인해 버튼을 다시 누를 수 있을 때까지 몇 분의 지연이 있을 것입니다.

## 사용자 필터링

`Search...` (검색...)이라고 표시된 검색 상자에서 사용자 필터링 및 검색을 수행할 수 있습니다.
사용자가 채널을 팔로우한 날짜와 표시 이름과 같은 모든 열 텍스트가 검색 가능합니다.
