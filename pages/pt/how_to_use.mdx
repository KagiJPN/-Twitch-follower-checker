## Autenticação Twitch

Na sua primeira visita, clique no botão `Authenticate with Twitch` para realizar a autenticação Twitch.
A lista exibida será baseada no usuário autenticado.

## Follower List

É exibida uma lista de usuários que atualmente estão seguindo seu canal.

## New followed List

É exibida uma lista de usuários que começaram a seguir seu canal desde a data do `Last checked`.

## Unfollowed List

É exibida uma lista de usuários que deixaram de seguir seu canal desde a data do `Last checked`.

## Atualização Diferencial

Ao pressionar o botão `CHECK DONE`, a data de `Last checked` será atualizada. Isso afeta a `New followed List` e a `Unfollowed List`.

## Atualizando as Listas

Pressione o botão `REFRESH LISTS` para recuperar os seguidores novamente e atualizar as listas.
Use esse recurso durante uma transmissão ao vivo ou sempre que quiser verificar as transições de seguidores em tempo real.
Devido às limitações da API, haverá um atraso de alguns minutos antes que você possa pressionar o botão novamente.

## Filtragem de Usuários

Você pode realizar a filtragem e busca de usuários a partir da caixa de pesquisa rotulada como `Search...`.
Todo o texto das colunas, como a data em que o usuário seguiu o canal e o nome de exibição, são pesquisáveis.
