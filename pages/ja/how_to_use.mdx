## Twitch 認証

初回アクセス時には、`Authenticate with Twitch` ボタンをクリックして Twitch 認証を行ってください。  
表示されるリストは認証を行ったユーザーを対象とします。

## Follower List

現在あなたのチャンネルをフォローしているユーザーの一覧が表示されます。

## New followed List

`Last checked` の日付の時点のフォロワーから新しくあなたのチャンネルをフォローしたユーザーの一覧が表示されます。

## Unfolloewd List

`Last checked` の日付の時点のフォロワーからあなたのチャンネルをフォロー解除したユーザーの一覧が表示されます。

## 差分更新
`Last checked` の日付を更新します。`New Follower List` と `Unfolloewd List` が対象です。

## リストを更新する

`REFRESH LISTS` ボタンを押下するとフォローしているユーザーを再度取得してリストを更新します。  
配信中などリアルタイムでフォロワーの推移を確認したい時に使います。  
API 制限があるため再度ボタンを押せるようになるまで数分設けています。

## ユーザーフィルタリング

`Search...` と書かれた検索窓からユーザーのフィルタリング・検索を行えます。  
チャンネルをフォローした日付や表示名など全て列の文字が対象になります。
